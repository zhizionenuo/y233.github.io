二维偏序

干什么

二维偏序处理的是这样一个问题：

对一个由二维组组成的集合 (a1,b1),(a2,b2),⋅⋅⋅,(an,bn){ (a^1,b_1),(a_2,b_2),···,(a_n,b_n) }(a1,b1​),(a2​,b2​),⋅⋅⋅,(an​,bn​) 进行偏序关系 ( 比如 (ai,bi)<(aj,bj){ (a_i,b_i)<(a_j,b_j)}(ai​,bi​)<(aj​,bj​) 定义为 ai<bi&& aj<bja_i<b_i \And\And\ a_j<b_jai​<bi​&& aj​<bj​ ） 处理

怎么做

将一个二维组拆分为 (x,y)

先按 x 排序,再比较 y 的大小。

 折叠——程序

没错，上面就是在求 1∼n1 \thicksim n1∼n 中有多少个数对满足大小关系。

可以发现暴力求一遍的 O(n2)O(n^2)O(n2) 作法似乎不大符合这个名字的逼格。

可以利用双指针和归并排序，将原代码降到 O(nlog⁡2n)O(n \log_2 n)O(nlog2​n)

题目背景

这是一道模板题，可以使用 bitset，CDQ 分治，KD-Tree 等方式解决。

题目描述

有 nnn 个元素，第 iii 个元素有 ai,bi,cia_i,b_i,c_iai​,bi​,ci​ 三个属性，设 f(i)f(i)f(i) 表示满足 aj≤aia_j \leq a_iaj​≤ai​ 且 bj≤bib_j \leq b_ibj​≤bi​ 且 cj≤cic_j \leq c_icj​≤ci​ 且 j≠ij \ne ij​=i 的 jjj 的数量。

对于 d∈[0,n)d \in [0, n)d∈[0,n)，求 f(i)=df(i) = df(i)=d 的数量。

接下来 nnn 行，每行三个整数 ai,bi,cia_i ,b_i,c_iai​,bi​,ci​，分别表示三个属性值。

输出格式

nnn 行，第 d+1d + 1d+1 行表示 f(i)=df(i) = df(i)=d 的 iii 的数量。

1≤n≤1051 \leq n \leq 10^51≤n≤105，1≤ai,bi,ci≤k≤2×1051 \leq a_i, b_i, c_i \le k \leq 2 \times 10^51≤ai​,bi​,ci​≤k≤2×105。
